## What is Sussurro?

Sussurro (Italian for "whisper") is a system-wide dictation tool that converts speech into clean, formatted text and injects it into any application — entirely on your machine. It uses **Whisper.cpp** for automatic speech recognition and a fine-tuned **Qwen 3** language model to clean up the output.

There is no cloud, no subscription, and no data ever leaves your device.

## How it works

When you press the global hotkey, Sussurro captures microphone audio, runs it through Whisper to get a raw transcription, then passes that text through Qwen 3 Sussurro — a fine-tuned model that removes filler words ("um", "uh"), handles self-corrections ("I want blue... no, red" → "I want red"), and applies grammar fixes. The result is written to the clipboard and pasted into the focused application.

A pill-shaped overlay capsule at the bottom of your screen shows the current state: idle, recording, or transcribing.

## Supported platforms

| Platform | Overlay UI | Global hotkey | System tray |
|----------|-----------|---------------|-------------|
| Linux X11 | GTK3 + Cairo | XGrabKey (automatic) | AppIndicator |
| Linux Wayland | GTK3 + layer-shell | External shortcut (DE-configured) | AppIndicator |
| macOS | Cocoa + CoreVideo | CGEventTap (automatic) | NSStatusItem |

## AI models

Sussurro requires two models, both downloaded on first run:

| Model | Size | Role |
|-------|------|------|
| Whisper Small | ~488 MB | Fast speech-to-text (default) |
| Whisper Large v3 Turbo | ~1.62 GB | Higher accuracy speech-to-text |
| Qwen 3 Sussurro (Q4_K_M) | ~1.28 GB | Text cleanup — always required |

You can switch Whisper models at any time from the Settings window or by running `sussurro --whisper`.

## Companion tool

**`sussurro-transcribe`** is a standalone CLI for transcribing audio files using the same local models. It accepts any format ffmpeg supports and optionally applies LLM cleanup. See [File Transcription](/guides/file-transcription).
